**Global Layoffs Analysis (SQL)**

**Objective**
Analyze global layoff trends to understand:
- Industries most affected by layoffs
- Countries with the highest layoff numbers
- Companies with the largest workforce reductions

**Tools**
- SQL (MySQL)

**Analysis Steps**
1. Data cleaning (remove duplicates, standardize values, handle nulls)
2. Exploratory Data Analysis (EDA)
3. Answering analytical questions

**Key Skills Demonstrated**
- Data cleaning using SQL
- Aggregation and grouping
- Trend analysis
- Analytical thinking
