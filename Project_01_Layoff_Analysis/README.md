# Global Layoff Analysis (SQL)

## Objective
Analyze global layoff data to identify:
- Industries most affected by layoffs
- Countries with the highest layoff numbers
- Companies with the largest workforce reductions

## Dataset
Public global layoff dataset covering multiple industries and countries.

## Tools
- SQL (MySQL)
- Tableau (Data Visualization)

## Analysis Steps
1. Data cleaning
2. Exploratory Data Analysis (EDA)
3. Analytical insights

## Key Skills Demonstrated
- Data cleaning using SQL
- Aggregation and grouping
- Trend analysis
- Analytical problem solving
